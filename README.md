# 二维太阳系模拟器

一个轻量级的二维太阳系演示项目，用原生 HTML、CSS 和 Canvas 展示太阳与八大行星的轨道运动。

## 项目信息

- 中文名：二维太阳系模拟器
- 英文标识：2d-solar-system-simulator
- 类型：静态网页 / Canvas 动画
- 入口文件：`index.html`

## 功能

- 太阳与八大行星二维轨道动画
- 速度调节
- 暂停 / 继续
- 重置模拟
- 行星标签开关
- 轨迹开关
- 点击行星查看轨道周期和平均距离

## 本地打开

直接用浏览器打开：

```text
C:\Users\敏\Documents\New project\index.html
```

也可以在项目目录中启动一个本地静态服务：

```bash
python -m http.server 4173 --bind 127.0.0.1
```

然后访问：

```text
http://127.0.0.1:4173/
```
